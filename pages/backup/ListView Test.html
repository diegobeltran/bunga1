Tap Behavior: Direct Select

When an item is clicked or tapped the item is selected and invoked (the ListView's oniteminvoked event will fire). If Selection Mode is set to None the item will not be selected.

                    
                    var listView = document.querySelector(".listView").winControl;
listView.tapBehavior = WinJS.UI.TapBehavior.directSelect;
